# 归档目录

此目录包含迁移过程中产生的临时文档，已被移动到这里以保持项目根目录的整洁。

## 目录结构

### migration-reports/
包含各种功能修复和优化的详细报告：
- FIX_REPORT_Label_Icon_Display.md - 标签图标显示修复
- LIST_DEFAULTITEM_FIX.md - 列表默认项修复
- SCROLLBAR_*.md - 滚动条相关修复
- STOP_PROPAGATION_FIX_SUMMARY.md - 事件传播修复

### migration-analysis/
包含迁移过程中的分析和设计文档：
- 组件状态报告
- Laya 兼容性分析
- 渲染管道设计
- API 迁移分析
- 性能优化提案
- 等等...

## 说明

这些文档在迁移过程中提供了重要的历史记录和设计思路，但不再是日常开发所需的文档。
如果需要查阅历史设计决策或问题解决方案，可以参考此目录。
